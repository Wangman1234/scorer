# Scorer

This is a chrome
[Isolated Web App](https://chromeos.dev/en/web/isolated-web-apps)
(IWA) for tournament scoring in HEMA. This application implements the 
[Cyrano protocol](https://superfencingsystem.com/CyranoProtocol-1-1.pdf)
for communicating with Fencing tournament software such as
[Fencing time](https://www.fencingtime.com/) for a simpler tournament 
running experience.
